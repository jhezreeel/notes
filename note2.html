My Notes / Reflection

Application Development (App Dev)

The challenging yet fulfilling process of building functional web apps.

Grabe, isa sa pinaka-“madugong” pero super enjoyable na subjects namin ay ang Application Development. Dito talaga na-feel ko kung gaano ako ka-dedicated sa paggawa ng functional web applications. Minsan nakakastress, lalo na kapag ang dami mong errors at kelangan i-debug ng paulit-ulit, pero ang saya ng feeling kapag finally gumana yung code mo after ilang beses mong binago at nireload. Parang ang rewarding talaga ng feeling na lahat ng effort mo sa bawat line of code, nagbabayad sa resulta.

Front-end: Bootstrap

Sa Bootstrap, natutunan namin gumawa ng clean at professional-looking front-end gamit yung mga built-in tools na sobrang easy gamitin pero ang linis ng output:

  • Navbar
  • Forms & Form Style Validation
  • Modal
  • Carousel
  • Tooltip
  • Popover

Ang best dito, kahit simple lang, ang daming pwedeng gawin para maging responsive ang design. Doon ko rin na-appreciate kung gaano ka-importante ang responsive design kasi gusto mo talaga na kahit anong device gamitin ng user, maganda at functional pa rin yung app mo.

Back-end: PHP Superglobals

Sa PHP naman, dito ko na-feel yung tunay na backend challenge — lalo na sa paggamit ng mga Superglobals:

  • $_GET
  • $_POST
  • $_SESSION
  • $_COOKIE
  • $_FILES

PHP Functions

Kasama pa d’yan ang Functions, na sobrang dami naming natutunan:

  • Defining Functions & Return Values
  • Function Arguments (Positional & Default)
  • Variable Scope (Global vs. Local)
  • Constants
  • Type Declarations
  • Anonymous Functions & Closures
  • Callback Functions
  • Arrow Functions

Bukod pa dito, nag-aral din kami ng Laravel at yung paggamit ng Databases gamit ang Laragon at Composer. Grabe, sa simula sobrang nakaka-overwhelm at challenging, parang “ang hirap talaga nito!” Pero once na gumana lahat ng codes at makita mo yung app na smooth at functional, sobrang fulfilling ng feeling. Parang reward sa effort mo sa bawat line ng code, at mas na-appreciate mo talaga yung process ng backend + frontend integration.

Hindi ko rin maikakaila na minsan AI tulong lang ang gamit ko sa pag-structure ng code o pag-debug — pero hindi para mandaya, kundi para matuto kung paano mas efficient at proper yung approach. Natutunan ko rin na patience at persistence talaga key sa programming, kasi kahit ilang beses mong palitan o i-reload yung code, kapag gumana, worth it lahat.

Overall, App Dev is both a challenge and a reward. Nakakapagod, pero sobrang saya at fulfilling. Ang saya kasi makita na gumagana yung web app na ikaw mismo ang gumawa — from the front-end design, backend logic, hanggang sa database at Laravel integration. Kahit mahirap, masasabi mong super worth it talaga every minute ng effort!
